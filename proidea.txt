# 🩺 Early Disease Prediction System — Full Project Context

## Project Overview
A machine learning system that predicts diseases from 24 normalized blood biomarkers. Takes patient lab values (0.0–1.0 range) as input and classifies them into one of 6 health conditions.

---

## Dataset
- **File:** `blood_samples_dataset_test.csv`
- **Shape:** 486 rows × 25 columns (24 features + 1 target)
- **Feature range:** All numeric features are normalized `0.0 – 1.0`
- **No missing values**
- **Target column:** `Disease` (string, truncated in raw CSV)

## ML Pipeline Architecture

### Step-by-step flow
```
Raw CSV
  → Clean disease name labels
  → LabelEncoder on target
  → Train/test split (80/20, stratified)
  → StandardScaler on features (fit on train only)
  → SMOTE on training set (k_neighbors=3, random_state=42)
  → Train 7 individual models
  → StratifiedKFold cross-validation (k=5)
  → Select top 3 by test accuracy
  → VotingClassifier ensemble (soft voting)
  → Compare ensemble vs best individual
  → Save best overall model
```

### Models Trained
```python
models = {

```

### Evaluation Metrics Used
- Accuracy (primary)
- 5-fold cross-validation (mean ± std)
- Classification report (precision, recall, F1 per class)
- Confusion matrix
- Feature importance (tree-based models)
- SHAP values (TreeExplainer or KernelExplainer fallback)

## Saved Model Artifacts
After training, 4 files are saved to `disease_prediction_model/`:

| File | Type | Purpose |
|------|------|---------|
| `model.pkl` | joblib | Final model (ensemble or best individual) |
| `scaler.pkl` | joblib | StandardScaler fitted on training data |
| `label_encoder.pkl` | joblib | LabelEncoder for target classes |
| `feature_names.json` | JSON | Ordered list of 24 feature names |

### Loading & Predicting (exact code pattern)
```python
import joblib, json, numpy as np

model  = joblib.load('disease_prediction_model/model.pkl')
scaler = joblib.load('disease_prediction_model/scaler.pkl')
le     = joblib.load('disease_prediction_model/label_encoder.pkl')
with open('disease_prediction_model/feature_names.json') as f:
    feature_names = json.load(f)

def predict(patient_dict: dict) -> dict:
    values = np.array([[patient_dict.get(f, 0.5) for f in feature_names]])
    scaled = scaler.transform(values)
    pred   = model.predict(scaled)[0]
    proba  = model.predict_proba(scaled)[0]
    return {
        'prediction'    : le.inverse_transform([pred])[0],
        'confidence'    : float(proba.max()),
        'probabilities' : dict(zip(le.classes_, proba.round(4)))
    }
```

## Key Design Decisions & Gotchas

| Decision | Reason |
|----------|--------|
| SMOTE with `k_neighbors=3` | Dataset has only 5 "Healthy" samples — default k=5 would fail |
| Stratified split | Maintains class proportions in tiny minority classes |
| Scaler fitted on train only | Prevents data leakage from test set |
| Soft voting ensemble | Averages probabilities, works better than hard voting when models are calibrated |
| `@st.cache_resource` | Prevents model reloading on every Streamlit interaction |
| Disease name cleaning | Raw CSV truncates names at 8 chars (e.g. `Thalasse`, `Heart Di`) |
| All features normalized 0–1 | No feature needs unit conversion before input |
